# 简历投递记录应用

## 项目介绍

这是一个基于 React 18 + TypeScript + Vite 开发的简历投递记录应用，专为毕业季学生设计，帮助用户全面记录和管理简历投递情况。

### 项目特点

- **全面的记录管理**：支持记录公司名称、岗位、官网URL、投递状态等信息，还可以添加自定义字段
- **自定义状态管理**：用户可以自定义投递状态，并为每个状态设置不同的颜色
- **Excel 导入/导出**：支持从 Excel 文件导入记录，也可以将记录导出为 Excel 文件
- **本地存储**：数据保存在用户本地，无需注册登录
- **简洁美观的界面**：使用 Tailwind CSS 构建，界面简洁美观，颜色使用不超过 4 种

### 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Zustand (状态管理)
- SheetJS (Excel 处理)

### 项目性质

**纯 AI 开发项目**：基于 Trae 的 SOLO 模式，由 AI 独立完成所有开发工作。

## 运行流程

### 1. 克隆项目

```bash
git clone <repository-url>
cd <project-directory>
```

### 2. 安装依赖

```bash
npm install
```

### 3. 启动开发服务器

```bash
npm run dev
```

应用将在 http://localhost:5173/ 启动

### 4. 构建生产版本

```bash
npm run build
```

构建后的文件将位于 `dist` 目录中

### 5. 检查 TypeScript 类型

```bash
npm run check
```

## 功能使用指南

1. **添加投递记录**：点击导航栏中的"添加记录"按钮，填写相关信息
2. **编辑记录**：点击记录卡片上的编辑图标
3. **删除记录**：点击记录卡片上的删除图标，会弹出确认对话框
4. **管理状态**：点击导航栏中的"状态管理"链接，添加、编辑或删除状态
5. **导入/导出**：使用导航栏中的导入/导出按钮处理 Excel 文件
6. **筛选记录**：使用首页的状态筛选栏按状态筛选记录

## 本地存储说明

- 应用使用 localStorage 存储数据，键名为 `job-application-tracker`
- 数据会在浏览器关闭后保留，除非用户手动清除浏览器数据
- 建议定期导出数据备份，以防数据丢失
- 每个浏览器的 localStorage 是独立的，更换浏览器或设备会导致数据不共享
